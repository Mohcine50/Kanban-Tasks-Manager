# Kanban-Tasks-Manager
Kanban Tasks Manager Using Mern Stack

## Technologies:
- React Js
- Redux Toolkit
- MongoDb
- Node Js / expressJs

## Features:

- Auth.
- Create a Global Board.
- Create task related to each tasks.
- Filter task by todo, doing and done.
